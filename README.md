# Vue QRCode Generator
A QRCode Generator made with Vue

## Screenshots
![Demo](demo/demo.gif)

## Dependencies
- VueJS
- Http Server
- QRious

## Dev dependencies
To run on dev environment run:

```sh
npm install
npm start
```
> The QRCode Generator will run on  http://localhost:8080
